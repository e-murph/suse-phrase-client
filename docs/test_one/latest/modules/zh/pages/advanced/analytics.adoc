= 分析页面
:revdate: 2025-07-10
:page-revdate: {revdate}
:description: SUSE Observability 自助托管

分析页面提供了一个强大而灵活的界面，可使用xref:/develop/reference/k8sTs-stql_reference.adoc[STQL 查询]来查询和分析可观测性数据。
这种交互式环境使您能够超越预先构建的仪表盘，深入了解动态环境的健康状况、性能和依赖性。

[CAUTION]
====
*分析页面已被弃用，但您仍可通过配置特定权限来启用它。不过，我们强烈建议利用 StackPacks 来增强可观察性和分析性，因为它们提供了一个更强大、更积极支持的替代方案。*

启用分析页面需要两个权限：

* `access-analytics`
* `execute-scripts`

更多详情：xref:/setup/security/rbac/rbac_permissions.adoc[权限管理]

====
