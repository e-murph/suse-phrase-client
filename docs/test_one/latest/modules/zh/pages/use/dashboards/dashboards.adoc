= 仪表板
:revdate: 2025-11-24
:page-revdate: {revdate}
:description: 创建和管理仪表板，使资源数据可视化。

仪表板可视化您的资源数据，并可根据您的需求创建和定制。仪表板可包含 IT 环境不同部分的资源。

== 创建新的仪表板

创建第一个仪表盘：

. 在主菜单中单击**仪表板**。
. 单击屏幕中间的**添加新仪表板**按钮。

要创建新的仪表板：

. 在主菜单中单击**仪表板**。
. 单击右上角的**添加新仪表板**按钮。
. 填写**名称**和**描述**。
. (可选）选择 "**公开**"，使所有用户都能看到仪表板。
. (可选）选择 "**星级**"可将仪表板添加到主菜单，以便快速访问。
. 单击**创建仪表板**。

您将被导航到新的空白仪表板。

image::dashboards/add-new-dashboard-drawer.png[创建新的仪表板抽屉]


== 仪表板属性

进入仪表盘后，使用顶部导航栏中*保存...*按钮旁边的下拉菜单并选择**编辑**仪表盘属性，就可以编辑仪表盘的属性。在这里，您可以更新名称和描述，以及仪表板是公开的还是星级的。

image::dashboards/edit-dashboard-properties-button.png[编辑仪表板属性位置]

image::dashboards/edit-dashboard-properties-drawer.png[编辑仪表板属性抽屉]

=== 公共仪表板
默认情况下，只有您可以查看仪表盘。将仪表板公开后，所有用户都能看到它。公共 dashbords 可供他人查看，但不可编辑。

=== 星级仪表板
给仪表板加星可使其在主菜单中以及从其他页面添加部件时显示得更突出。当您想快速访问仪表板时，可将其设为星形。

星级仪表盘的范围针对用户。其他用户有自己的星级仪表盘。

您还可以在仪表盘名称旁边的星形图标处给仪表盘加星或取消加星。

== 仪表板设置

仪表盘设置可让您控制时间范围、数据刷新间隔和变量是否保存在仪表盘中，或由查看器自定义。单击仪表板右上角的齿轮图标打开仪表板设置。

启用 "保存..."标题下的某个复选框（例如时间范围）后，仪表盘将在保存后固定为所选的时间范围。禁用时，时间范围可由仪表盘的查看者选择；更改会反映在 URL 中，因此可以与其他人共享链接，让他们以相同的时间范围查看仪表盘。
