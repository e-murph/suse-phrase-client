:description: 在仪表板上组织、调整、编辑和删除部件。

== 排列和调整部件大小

您可以在仪表板上组织部件，创建适合自己的布局。

* 要移动 Widget，请单击并拖动 Widget 的标题区域到新位置。
* 要调整 Widget 的大小，请单击并拖动 Widget 的右下角。

移动或调整部件大小时，仪表板上的其他部件会自动重新排列。

== 编辑或删除小工具

. 要编辑或删除小工具，请单击小工具右上方的更多操作图标。

. 选择 "*编辑 Widget "*或 "*删除 Widget*"。

要更详细地检查部件的数据，请单击度量检查器抽屉中 "更多操作 "按钮旁边的图标。
