= 自定义视图
:revdate: 2025-07-10
:page-revdate: {revdate}
:description: SUSE 可观察性

== 概述

xref:/use/views/k8s-views.adoc[kubernetes 视图]是探索 IT 环境特定部分的起点，xref:/use/views/k8s-explore-views.adoc[探索视图]允许您调查特定视图的子集，而*自定义*视图则为您提供了返回这些视图中任何一个的方法。

换句话说，您可以通过保存xref:/use/views/k8s-views.adoc[kubernetes 视图]或带有自己设置（筛选器、可视化选项、视图设置、时间线配置）的xref:/use/views/k8s-explore-views.adoc[探索视图]来创建自定义视图，从而将您或您的团队特别感兴趣的拓扑结构的一部分添加到书签中。

[NOTE]
====
默认情况下，所有用户都可看到自定义视图，如有需要，*SUSE Observability 管理员*可对这些视图进行加密。
====


image::k8s/k8s-custom-view.png[自定义视图]

== 处理

所有自定义视图都可以在主菜单的 "*视图*"条目下找到。

如果您经常使用某个自定义视图，可以将其添加到*星级视图*中。所有星级视图都直接列在主菜单的 "*视图*"部分，以便快速访问\--您可以通过名称旁边的黄色星形图标识别星级视图。只需点击自定义视图名称旁边的星形图标，即可从星形视图中添加或删除该视图。

=== 创建视图

要创建新的自定义视图，请在xref:/use/views/k8s-views.adoc[kubernetes 视图]或xref:/use/views/k8s-explore-views.adoc[探索视图]上单击顶部导航栏上的 "*将视图另存为...*"蓝色按钮。要根据修改后的自定义视图创建新视图，请使用按钮旁边的下拉菜单并选择将*视图另存为...* 。

image::k8s/k8s-custom-view-edit-settings.png[编辑视图设置]

在 "*视图另存为*"对话框中，可以设置以下选项：

|===
| 字段名称 | 说明

| 查看名称
| 视图的名称。

| 说明
| 景观简述这有助于记住视图的范围。

| 标识符
| 视图的唯一引用，有助于从导出配置（如 StackPack 中的导出配置）中引用视图。
|===

=== 重置视图

创建自定义视图后，所有筛选器、可视化选项、视图设置和时间线配置都会保存在视图中。如果您想在对自定义视图进行更改后将其重置为原始状态，请使用顶部导航栏上*保存...*按钮旁边的下拉菜单并选择*重置视图*。

=== 删除或编辑视图

[NOTE]
====
请注意，对自定义视图所做的更改将应用于所有用户。
====


自定义视图可以在 "*所有视图*"屏幕或 "*自定义视图*"屏幕上编辑或删除。

. *所有视图*屏幕：
 ** 单击主菜单中的 "*视图*"打开*所有视图*屏幕。
 ** 将鼠标悬停在要编辑或删除的视图上。
 ** 单击右侧的*编辑视图设置*或*删除视图*按钮。
. *自定义视屏*：
 ** 打开视图。
 ** 打开顶部导航栏上*保存...*按钮旁边的下拉菜单。
 ** 单击*编辑视图设置*或*删除视图*。

== 结构

自定义视图的结构与xref:/use/views/k8s-views.adoc[kubernetes 视图]或根据其创建的xref:/use/views/k8s-explore-views.adoc[探索视图]完全相同：具有相同的xref:/use/views/k8s-view-structure.adoc#_filters[过滤器]和xref:/use/views/k8s-view-structure.adoc#_perspectives[视角]。
