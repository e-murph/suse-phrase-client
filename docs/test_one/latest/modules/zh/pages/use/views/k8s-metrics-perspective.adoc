== 图表

图表近乎实时地显示所选组件的指标数据\--每 30 秒获取一次数据。如果某个进程停止运行，并且不再接收数据，那么该进程最终将离开图表，因为数据至少每 30 秒左右会向左移动一次。如果在 30 秒的时间间隔内收到更多数据，这些数据将被推送到图表中。
