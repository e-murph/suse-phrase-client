= 概览视角
:revdate: 2025-07-10
:page-revdate: {revdate}
:description: SUSE 可观察性

概览视角显示视图中所有组件的列表。根据视图类型和视图中组件的不同，概览视角的结构也会不同。

例如，用于xref:/use/views/k8s-views.adoc[kubernetes 视图]的表结构将反映每个视图中包含的组件类型的最重要属性：如下所示，`services` 视图的表结构与`pods` 视图不同。

image::k8s/k8s-overview-perspective-table-comparison.png[概览表结构比较]

对于xref:/use/views/k8s-custom-views.adoc[自定义视图]和xref:/use/views/k8s-explore-views.adoc[探索视图]，由于视图中可能包含多种组件类型，因此概览透视表将采用通用的 "一刀切 "结构，由最常见的属性组成。

image::k8s/k8s-overview-perspective-generic-table.png[概览表通用结构]

虽然概览视角的大多数视图类型都使用表格布局，但在某些情况下也会提供卡片布局，让您可以在不同的概览视角内容显示模式之间进行切换。

image::k8s/k8s-overview-perspective-cards-layout.png[卡片布局概览]
