= 活动视角
:revdate: 2025-07-10
:page-revdate: {revdate}
:description: SUSE 可观察性

== 概述

事件视角显示当前xref:/use/views/k8s-view-structure.adoc[视图]或筛选拓扑中元素的事件和更改。

image::k8s/k8s-events-perspective.png[活动视角]

== 筛选显示的事件

显示的事件可在视图用户界面右上角的筛选*器*菜单中进行筛选。事件筛选器将应用于视图中显示的所有事件：事件视图、亮点视图和时间线。

=== 按属性筛选

在*筛选器*菜单中设置的xref:/use/views/k8s-filters.adoc#_filter_events[事件筛选器]可用于根据事件类别、类型、源系统和标记对显示的事件进行细化。

=== 按源组件筛选

在*筛选器*菜单中设置的xref:/use/views/k8s-filters.adoc#_filter_topology[拓扑筛选器]可定义显示事件的元素（组件和关系）。只有与应用的拓扑过滤器或视图本身相匹配的元素相关的事件才会可见。您可以通过更新拓扑筛选器来调整显示事件的组件。

=== 按时间戳筛选

事件视角和事件列表显示与视图用户界面底部时间线中选择的xref:/use/stackstate-ui/k8sTs-timeline-time-travel.adoc#_telemetry_interval[遥测时间间隔]相匹配的事件。调整遥测时间间隔，以便只显示该时间段内发生的事件。

== 时间旅行

事件视角列出了已生成的事件：

* 在xref:/use/stackstate-ui/k8sTs-timeline-time-travel.adoc#_topology_time[拓扑时间]指定的时间点上存在的拓扑元素。
* 在选定的xref:/use/stackstate-ui/k8sTs-timeline-time-travel.adoc#_telemetry_interval[遥测时间间隔]内

这样，您就可以穿越时空，查看历史上某个特定时间点的拓扑元素所产生的事件，以及某个特定时间窗口内产生的事件。选择一个新的拓扑时间，就会穿越时空回到当时可用的拓扑。事件列表将被更新，以包括当时可用的拓扑元素和所选遥测时间间隔内的事件。

例如

* 调整*遥测间隔*以增加或减少显示的事件数量。
* 调整 "*拓扑*"*时间*，穿越时空回到当时可用的拓扑。将显示拓扑时间内存在的拓扑元素在所选遥测时间间隔内产生的事件。
* 点击事件*属性*中的时间戳，跳转到该拓扑时间。这将更新事件列表，显示已生成的事件：
 ** 由当时存在的拓扑元素决定。
 ** 在指定的遥测时间间隔内（如果需要，将根据所选时间戳进行调整）。
