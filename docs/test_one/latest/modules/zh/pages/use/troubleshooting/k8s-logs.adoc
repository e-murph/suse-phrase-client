== 概述

Kubernetes 会保存集群活动和应用程序输出的全面日志。日志记录为了解容器、节点和 Kubernetes 本身的运行情况提供了宝贵的信息。它能细致地记录从 Pod 到 ReplicaSets 的所有信息，让您可以追溯问题的源头。

== 访问 pod 日志

日志中的错误和警告表明应用程序、容器或 pod 本身存在潜在问题。要在 pod 屏幕上查看日志，请单击*显示*日志按钮。

== 使用 pod 日志

您可以使用搜索功能查找日志中的特定错误或警告。日志可通过容器过滤。您可以选择所需的日志显示时间间隔以及日志更新频率的刷新间隔。

日志直方图可直观显示选定时间间隔内的日志行数，并可通过单击和拖动放大到特定时间间隔。

使用这些选项可显示或隐藏每行日志的附加信息：

* *显示时间*以显示时间戳
* *显示容器*，以显示日志行所属的容器名称
* *缠绕线条*，使长线条能够缠绕
* *最新先入/最旧先入*可切换日志排序顺序
