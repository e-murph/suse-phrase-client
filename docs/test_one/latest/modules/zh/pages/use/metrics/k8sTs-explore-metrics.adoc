= 探索衡量标准
:revdate: 2025-07-10
:page-revdate: {revdate}
:description: SUSE 可观察性

您可以在 SUSE Observability 主菜单底部找到度量资源管理器。使用它可以执行任何 PromQL 查询，并可视化由此产生的时间序列。查询结果显示在所选时间范围的图表和表格中，表格显示最后一个值以及时间序列的标签。

image::k8s/k8s-metrics-explorer.png[度量资源管理器]

== PromQL 查询

ifdef::ss-ff-stackpacks2_enabled[]
查询输入框会自动推荐度量名称、标签名称和值，以及支持的 PromQL 函数。有关完整的https://prometheus.io/docs/prometheus/latest/querying/basics/[PromQL 指南和参考]，请参阅 Prometheus 文档。SUSE Observability 还添加了 2 个默认参数，可在任何查询中使用：`${__interval}` 和`${__rate_interval}` 。它们可用于根据图表分辨率自动缩放汇总区间xref:/setup/custom-integrations/metric-bindings/writing-promql.adoc[（更多详情]）。
endif::[]

ifndef::ss-ff-stackpacks2_enabled[]
