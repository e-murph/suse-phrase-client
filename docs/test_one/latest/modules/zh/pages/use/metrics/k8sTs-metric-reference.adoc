== 概述

本页定义了 SUSE Observability 收集的常见 Kubernetes 相关指标，并解释了如何计算这些指标。

=== container_memory_usage

[horizontal]
名称:: `container_memory_usage`
类型:: gauge
单位:: 字节数
资料来源:: 直接从容器 cgroup 层次结构中读取。

[cols="1,3"]
|===
|cgroup 版本 | 源文件

|v1 | `memory.usage_in_bytes`
|v2 | `memory.current`
|===

摘要:: cgroup 及其后代当前使用的内存总量。这个总和包括匿名内存 (RSS)、页面缓存和内核内存。使用以下指标之一来获取特定内存区域的更多详细信息。

=== container_memory_kernel

[horizontal]
名称:: `container_memory_kernel`
类型:: gauge
单位:: 字节数
资料来源:: 直接从容器 cgroup 层次结构中读取。

[cols="1,3"]
|===
|cgroup 版本 | 源文件

|v1 | `memory.kmem.usage_in_bytes`
|v2 | `memory.stat` -> `kernel` 文件内的字段
|===
摘要:: 内核内存总量，包括（kernel_stack、pagetables、percpu、vmalloc、slab）

=== container_memory_rss

[horizontal]
名称:: `container_memory_rss`
类型:: gauge
单位:: 字节数
资料来源:: 直接从容器 cgroup 层次结构中读取。

[cols="1,3"]
|===
|cgroup 版本 | 源文件

|v1 | `memory.stat` -> `total_rss` 文件内的字段
|v2 | `memory.stat` -> `anon` 文件内的字段
|===
摘要:: 匿名内存的驻留集大小 (RSS)。这是 cgroup 内的进程正在使用的非文件支持内存（堆和栈）。该值通常反映了应用程序的真实 "工作集"。

=== container_memory_cache

[horizontal]
名称:: `container_memory_cache`
类型:: gauge
单位:: 字节数
资料来源:: 直接从容器 cgroup 层次结构中读取。

[cols="1,3"]
|===
|cgroup 版本 | 源文件

|v1 | `memory.stat` -> `total_cache` 文件内的字段
|v2 | `memory.stat` -> `file` 文件内的字段
|===
摘要:: 页面缓存（用于缓存文件系统数据的内存，包括 tmpfs 和共享内存）。
