== 配置团队通知

要向 Slack 发送通知，请按以下步骤操作：

. <<_create_a_power_automate_flow,创建 Power Automate 流程>>
. <<_add_and_test_the_channel,添加并测试通道>>

=== 创建 Power Automate 流程

在 Teams 中，从 "Webhook "模板创建一个新流程。

image::k8s/notifications-teams-webhook-template.png[从 Webhook 模板创建流程]

选择要粘贴通知的团队和频道，然后保存流程。

编辑流程并单击 "收到团队 webhook 请求时 "框。
复制 HTTP URL 参数。

image::k8s/notifications-teams-select-url.png[从流程中选择 URL]

== 添加并测试通道

image::k8s/configure-teams-channel.png[配置团队频道]

回到 SUSE Observability，您现在可以使用 Webhook URL 创建通知通道。

== 团队消息通知

打开或关闭通知时，会在频道中创建一条新的 Teams 消息。
