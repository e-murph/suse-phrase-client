= 通知
:revdate: 2025-07-10
:page-revdate: {revdate}
:description: SUSE 可观察性

SUSE Observability 中的通知是在 SUSE Observability 中的xref:/use/alerting/k8s-monitors.adoc[监控器]触发时通知第三方应用程序的方法。通知可与第三方应用程序集成，如事件管理工具（如 PagerDuty 或 Opsgenie）、票据系统（如 JIRA、ServiceNow）或其他协作平台（如 Slack）。其他工具可能将通知称为警报。

SUSE Observability 组件上监控器的健康状态变化会触发通知。通过配置的通知通道，健康、组件和监控信息将被发送到这些外部系统。
